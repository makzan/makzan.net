Books I wrote

I share game design and HTML5 techniques in books. I’m now writing the 2nd edition of HTML5 Games Development by Examples: Beginner’s guide.

HTML5 Games Development by Examples: Beginner’s guide—Second Edition

Step-by-step examples to guide getting start making games with the latest web technology. You learn to create several HTML5 games from scratch. By the end of the book, you'll have the knowledge, skills, and level of understanding you need to efficiently develop games over the network using HTML5.

HTML5 Game Development Hotshot

You learn to create 8 HTML games by combining different web standard technologies. The book is project-based style and you will have 8 games ready for you to modify and publish them.

Video Course: HTML5 Games Development

Practical screencast-based tutorials that show you how to create two simple yet elegant games in HTML5. Presented in a modular approach, with elegant code and illustrated concepts to help you learn quickly.

Flash Multiplayer Virtual Worlds

Through the book, it shows how an online virtual world is built with socket server connection. It includes detail isometric view construction which can apply to flash games or other platform's games.
